# Техническое задание

## 1. Описание задачи
Необходимо разработать консольную программу, которая демонстрирует работу библиотеки orjson для сериализации и десериализации данных. Особое внимание уделяется работе с типом datetime, который часто используется в реальных приложениях.

## 2. Цель работы
Цель работы — изучить процесс преобразования Python-объектов в формат JSON и обратно, а также понять преимущества библиотеки orjson по сравнению со стандартным модулем json.

## 3. Функциональные требования
Программа должна:
- создать структуру данных (словарь), содержащую строку и объект datetime  
- выполнить сериализацию данных в формат JSON  
- вывести результат сериализации  
- выполнить десериализацию JSON обратно в Python-объект  
- вывести результат десериализации  

## 4. Нефункциональные требования
- код должен быть понятным и читаемым  
- программа должна корректно обрабатывать тип datetime  
- операции должны выполняться быстро  
- вывод должен быть информативным  

## 5. Ограничения и допущения
- используется небольшой объём данных  
- программа работает в консоли  
- не используется ввод пользователя  
- результат не сохраняется в файл
