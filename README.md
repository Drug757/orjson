# orjson 

## 1. Краткое описание инструмента
orjson — это библиотека Python для быстрой сериализации и десериализации данных в формате JSON. Она отличается высокой производительностью и поддерживает дополнительные типы данных, такие как datetime.

## 2. Цель изучения
Целью было изучить процесс преобразования данных в JSON и обратно, а также понять преимущества orjson по сравнению со стандартным модулем json.

## 3. Описание реализованной задачи
В рамках проекта была реализована программа, которая:
- создаёт словарь с данными  
- включает объект datetime  
- преобразует данные в JSON  
- выполняет обратное преобразование  

Особенностью является то, что datetime обрабатывается автоматически без дополнительного кода.

## 4. Инструкция по запуску
1. Установить библиотеку:
   ```
   pip install orjson
   ```
2. Запустить программу
   ```
   python main.py
   ```

## 5. Общие выводы
Библиотека orjson показала себя как быстрый и удобный инструмент для работы с JSON. Особенно полезной оказалась встроенная поддержка datetime, которая упрощает разработку.
